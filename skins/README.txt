把 *.json 皮肤文件放到这个 skins\ 文件夹，重开设置窗口即可在「主题」下拉里看到。

字段说明：
  name       皮肤名（显示在下拉里，必填）
  font       字体名，如 "Consolas" / "Segoe UI Light" / "微软雅黑"
  bold       true / false 是否加粗
  timeColor  时:分:秒 颜色，#RRGGBB
  dateColor  星期/日期 颜色，#RRGGBB
  back       背景：""=不改 | "glass"=玻璃 | "#RRGGBB"=纯色

示例见 我的紫色皮肤.json
